Krum Function

Here's how to use the Krum function:


                # Import the library
                import charisius

                # Initialize aggregator and vectors
                aggregator = ...
                vectors = [...]

                # Call krum function
                result = charisius.krum(aggregator, vectors)

Inputs

  • aggregator: object - The aggregator object containing the 'nb_byz' property.
  • vectors: list - The list of PyTorch Tensors to be processed.

Outputs

  • result: torch.Tensor - The vector with smallest Krum score.
