MoNNA Function

Here's how to use the MoNNA function:


                # Import the library
                import charisius

                # Initialize aggregator and vectors
                aggregator = ...
                vectors = [...]

                # Call MoNNA function
                result_old = charisius.MoNNA_old(aggregator, vectors)
                result_new = charisius.MoNNA(aggregator, vectors)

Inputs

  • aggregator: object - The aggregator object containing the 'nb_byz' property.
  • vectors: list - The list of PyTorch Tensors to be processed.

Outputs

  • result_old, result_new: numpy.ndarray or torch.Tensor - The result after applying the MoNNA function.
