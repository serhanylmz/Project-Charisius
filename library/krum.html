Krum Function

Here's how to use the Krum function:


                # Import the library
                import charisius

                # Initialize data
                data = [...]

                # Call Krum function
                result = charisius.krum(data, f)

Inputs

  • data: array - The data to be processed.
  • f: integer - The parameter to modify the function's behavior.

Outputs

  • result: array - The processed data.
