Geometric Median Function

Here's how to use the Geometric Median function:


                # Import the library
                import charisius

                # Initialize vectors
                vectors = [...]

                # Call geometric_median function
                result_old = charisius.geometric_median_old(_, vectors)
                result_new = charisius.geometric_median(_, vectors)

Inputs

  • vectors: list - The list of PyTorch Tensors to be processed.

Outputs

  • result_old, result_new: torch.Tensor - The geometric median of the input vectors.
