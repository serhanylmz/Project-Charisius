Multi Krum Function

Here's how to use the Multi Krum function:


                # Import the library
                import charisius

                # Initialize aggregator and vectors
                aggregator = ...
                vectors = [...]

                # Call multi_krum function
                result_old = charisius.multi_krum_old(aggregator, vectors)
                result_new = charisius.multi_krum(aggregator, vectors)

Inputs

  • aggregator: object - The aggregator object containing the 'nb_byz' property.
  • vectors: list - The list of PyTorch Tensors to be processed.

Outputs

  • result_old, result_new: torch.Tensor - The average of the k vectors with lowest scores.
