Bucketing Function

Here's how to use the Bucketing function:


                # Import the library
                import charisius

                # Initialize aggregator and vectors
                aggregator = ...
                vectors = [...]

                # Call bucketing function
                result_old = charisius.bucketing_old(aggregator, vectors)
                result_new = charisius.bucketing(aggregator, vectors)

Inputs

  • aggregator: object - The aggregator object containing the 'nb_byz' property and 'bucket_size'.
  • vectors: list - The list of PyTorch Tensors to be processed.

Outputs

  • result_old, result_new: torch.Tensor - The result after applying the bucketing function.
